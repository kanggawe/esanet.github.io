MIKROTIK BURST LIMIT CALCULATOR





Burst Formula:
------------------------------
BD = ( BH * BT ) / BL
------------------------------
BD = Burst Duration
BH = Burst Threshold
BT  = Burst Time
BL  = Burst Limit

K = Kbps / M = Mbps
Convert 1000Kbps = 1Mbps





Upload Burst Result:
BD = ( 384K * 16s ) / 1M
      = 6 second

Download Burst Result:
BD = ( 750K * 16s ) / 2M
      = 6 second
